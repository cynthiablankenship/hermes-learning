# ❓ FAQ

## General

**Q: Do I need to know math to learn this?**
A: No. This curriculum is designed to teach you the practical skills first. Math concepts are introduced only when they're directly relevant to understanding what you see in logs and tools.

**Q: How long should each module take?**
A: Module 1-2 can be done in a few days each if you spend 1-2 hours per day. Later modules are deeper and may take a week each. There's no rush.

**Q: What if I get stuck?**
A: Ask Hermes. That's what I'm here for. Any question is fair game — no such thing as a dumb question in this curriculum.

**Q: Do I need access to a DGX to follow along?**
A: For most modules, no. You can practice Linux and Docker on any machine. For DGX-specific lessons, I'll mark what requires actual hardware access.

## Technical

**Q: Windows or Linux?**
A: Most AI infrastructure runs on Linux. This curriculum assumes Linux. If you're on Windows, you can use WSL2 (Windows Subsystem for Linux) or a VM for practice.

**Q: What editor should I use?**
A: For now, `nano` for quick edits in the terminal. Later we'll look at VS Code. Don't worry about this yet.

**Q: Why are there so many different NVIDIA tools?**
A: NVIDIA has built an entire ecosystem around GPUs. Each tool does a specific job. We'll cover them one at a time so it doesn't feel overwhelming.
